C BASICS


C:

C is a general-purpose, procedural programming language.Developed by Dennis Ritchiein 1972 at Bell Labs.It is widely used for system programming like operating systems, compiler, etc. It is a compiled language and offers low-level memory access.

Structure of a C Program:


Syntax:

  • #include < stdio.h >

  • }

Names used for variables, functions, arrays, etc. Must start with a letter or underscore. Also case-sensitive.
