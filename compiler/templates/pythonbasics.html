PYTHON BASICS



Python:


Python is a high-level, interpreted, general-purpose programming language created by Guido Van Rossum in 1991.It emphasizes readable,concise code with a straightforward syntax and supports multiple programming paradigms(procedural, object-oriented, and functional programming).Python is dunamically typed and garbage-collected, making it beginner-friendly and highly productive for rapid application development.

  1. Easy to Learn & Readable
  2. Cross-Platform
  3. Extensive Libraries
  4. Community Support
  5. Multi-Purpose

Python Syntax with Definitions & Examples


1. Comments


Definition:

Used to explain code; not executed by Python.


Examples:

# This is a single-line comment
""" This is a multi-line comment """


2. Variables & Data Types


Definition:

Variables store data; data types describe the kind of data.


Examples:

age = 21 # int (integer)
pi = 3.1415 # float (decimal number)
name = "Alice" # str (string/text)
is_student = True # bool (True or False)


3. Type Conversion (Casting)


Definition:

Convert data from one type to another.


Examples:

num = int("123") # converts string to int
flt = float(10) # converts int to float
txt = str(456) # converts int to string


4. Input and Output


Definition:

Read input from the user and output.


Examples:

name = input("Your name: ") # read input as string
print("Hello,", name) # output


5. Operators


Definition:

Used to do calculations and logical comparisons.
Arithmetic: + - * / // % **
Comparison: == != > < >= <=
Logical: and or not
Assignment:+= -= *= /= %=
Conditional:?:

Examples:

3 + 2 # 5
5 == 5 # True
not True # False


6. Conditionals (if-elif-else)


Definition:

Run code only if a condition is true.


Examples:

if age >= 18:
print("Adult")
elif age >= 13:
print("Teenager")
else:
print("Child")


7. Loops


Definition:

Repeat a block of code multiple times.


Examples:

# for-loop
for i in range(3):
print(i) # prints 0,1,2

# while-loop
n = 0
while n < 3:
print(n)
n += 1


8. Functions


Definition:

Reusable blocks of code that do a task.


Examples:

def greet(name):
return f"Hello, {name}!"
print(greet("Alice"))


9. Lists (Mutable Ordered Collection)


Definition:

Ordered sequence that can hold different data types.


Examples:

fruits = ["apple", "banana"]
fruits.append("mango") # add to list
print(fruits[0]) # access elements


10. Tuples (Immutable Ordered Collection)


Definition:

Ordered sequence that cannot be changed after creation.


Examples:

coordinates = (10, 20)
print(coordinates[0]) # access elements


11. Sets (Unordered Unique Elements)


Definition:

Collection of unique elements.


Examples:

numbers = {1, 2, 2, 3}
numbers.add(4) # add element
numbers.remove(2) # remove element


12. Dictionaries (Key-Value Pairs)


Definition:

Stores data as pairs (key: value).


Examples:

person = {"name": "Alice", "age": 25}
person["age"] = 26
print(person.keys()) # dict_keys(['name', 'age'])


13. Strings


Definition:

Sequence of characters.


Examples:

s = "Hello"
s.upper() # 'HELLO'
s.lower() # 'hello'
s.replace('H', 'J') # 'Jello'


14. Slicing (Get Part of a Sequence)


Definition:

Extract sub-sequences from lists, tuples, strings.


Examples:

text = "Python"
print(text[1:4]) # 'yth'
print(text[:2]) # 'Py'
print(text[::-1]) # 'nohtyP'


15. List Comprehensions


Definition:

Create new lists by applying an expression to each element.


Examples:

squares = [x*x for x in range(5)] # [0, 1, 4, 9, 16]
evens = [x for x in range(10) if x%2==0] # [0, 2, 4, 6, 8]


16. Error Handling (try/except)


Definition:

Catch and handle errors gracefully.


Examples:

try: num = int("Hello") # will raise ValueError
except ValueError as e:
print("Error:", e)
finally:
print("Done!")


17. Importing Modules


Definition:

Include external functionality.


Examples:

import math
print(math.sqrt(16)) # 4.0

from math import pi
print(pi) # 3.14159...


18. Classes and Objects (OOP)


Definition:

Blueprints for creating objects with attributes and methods.


Examples:

class Dog:
def __init__(self, name):
self.name = name
def bark(self):
print(self.name + " says woof!")
dog = Dog("Buddy")
dog.bark() # Buddy says woof!


19. File Handling


Definition:

Read from and write to files.


Examples:

# Write to a file
with open('file.txt', 'w') as f:
f.write('Hello World')

# Read from a file
with open('file.txt', 'r') as f:
content = f.read()
print(content)


20. Lambda Functions (Anonymous Functions)


Definition:

Quick, one-line functions without a name.


Examples:

square = lambda x: x * x
print(square(5)) # 25
