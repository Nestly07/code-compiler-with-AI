JAVA BASICS


Java:

Java is a high-level, object-oriented, platform-independent programming language developed by James Gosling at Sun Microsystems(now owned by Oracle).

Write Once,Run Anywhere(WORA): Java programs can run on any system with a Java Virtual Machine(JVM).
